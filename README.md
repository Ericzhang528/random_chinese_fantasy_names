# random_chinese_fantasy_names

随机生成人名、法号、地名、门派名、法宝名等。

这是个[河图脚本语言](https://github.com/hetu-script/hetu-script)的库。

## 使用方法

安装河图脚本运行库：

```
dart pub global activate hetu_script_dev_tools
```

在本项目目录下输入：

```
hetu random_names.ht
```

进入河图命令行后，输入函数名字获得随机名：

```
getRandomName()
```
